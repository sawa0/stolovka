# Система управления столовой

Веб-приложение для управления заказами, меню, блюдами и отчетностью столовой.

## Возможности

- **Управление пользователями**: добавление, редактирование, активация/деактивация
- **Управление меню**: создание недельного меню, регулярное меню
- **Управление блюдами**: создание блюд, рецептов с ингредиентами, автоматический расчет себестоимости
- **Управление ингредиентами**: добавление, редактирование цен, единицы измерения
- **Система заказов**: прием заказов с подтверждением или автоматически
- **Отчетность**: генерация Excel-отчетов по заказам и рецептам
- **Telegram интеграция**: автоматическая отправка отчетов в Telegram

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python stolovka.py
```

3. Откройте в браузере:
- Главная страница (заказы): http://127.0.0.1:8080/
- Страница повара: http://127.0.0.1:8080/povar
- Настройки: http://127.0.0.1:8080/config

## Конфигурация

Основные настройки находятся в `config.py`:

- `HOST` - хост сервера (по умолчанию: 0.0.0.0)
- `PORT` - порт сервера (по умолчанию: 8080)
- `DEBUG` - режим отладки (по умолчанию: False)
- `PRICE_MARKUP` - наценка на блюда (по умолчанию: 1.11 = 11%)
- `REPORT_SCHEDULE_*` - настройки автоматической отправки отчетов

## База данных

Приложение использует SQLite с следующими таблицами:

- `users` - пользователи системы
- `dishes` - блюда с рецептами и ценами
- `products` - ингредиенты
- `transactions` - история заказов
- `menu` - недельное меню
- `settings` - настройки приложения

## Функции

### Расчет себестоимости блюд

Себестоимость блюда рассчитывается автоматически на основе:
- Количества каждого ингредиента в рецепте
- Цены ингредиентов
- Наценки (по умолчанию 11%)

При изменении цены ингредиента автоматически пересчитываются цены всех блюд, содержащих этот ингредиент.

### Отчеты

Система генерирует два типа Excel-отчетов:
1. **Отчет по заказам** - детализация заказов по пользователям и датам
2. **Отчет по рецепту** - состав блюда, себестоимость, общий объем ингредиентов

### Автоматическая отправка отчетов

Настраивается в разделе "Настройки":
- Отчет за прошедший месяц отправляется автоматически 1-го числа в 12:00
- Требуется указать Telegram Bot API Key и User ID

## Технологии

- **Backend**: Python 3, Flask, Flask-SocketIO
- **Database**: SQLite3
- **Frontend**: HTML, CSS, JavaScript, Socket.IO
- **Reports**: openpyxl
- **Scheduler**: APScheduler
